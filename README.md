# Varus Product Matcher

Система для автоматичного підбору реальних товарів із мережі **Varus** на основі списку покупок. Проєкт використовує **SerpApi** для отримання актуальних результатів пошуку Google та **GitHub Models/OpenAI API (GPT-4o mini)** для інтелектуального вибору найкращого відповідника.

---

## Основні можливості

* **Пошук через Google Search API**: Отримання реальних посилань на товари безпосередньо з домену `varus.ua`.
* **ШІ-фільтрація**: Використання LLM для відбору найбільш релевантних позицій (пріоритет свіжим продуктам над замороженими чи сушеними).
* **Автоматичне очищення даних**: Видалення рекламних міток (`srsltid`) та маркетингових заголовків із назв.
* **Smart Matching**: Визначення місткості упаковки та відповідності запиту користувача.
* **Експорт у CSV**: Формування готового звіту з назвами продуктів та прямими посиланнями.

---

## Архітектура

Проєкт реалізований за допомогою двох основних модулів:

### 1. `SerpApiVarusSearcher`
* Виконує запити до **SerpApi (Google Engine)**.
* Обмежує область пошуку лише сайтом `varus.ua`.
* Очищує URL від параметрів відстеження та фільтрує технічні сторінки (кошики, результати внутрішнього пошуку).

### 2. `AIProductMatcher`
* Отримує список знайдених посилань та запит користувача.
* Використовує промпт-інжиніринг для вибору конкретної одиниці товару.
* Повертає структуровані дані (JSON) через модель **GPT-4o mini**.

## Встановлення

### 1. Клонування репозиторію
```bash
git clone https://github.com/D3cryptHAX/Varus_ProductMatcher
cd Varus_ProductMatcher
```

### 2. Встановлення залежностей
```bash
pip install httpx python-dotenv openai pydantic
```

### 3. Налаштування `.env`
Створіть файл `.env` у кореневій папці та додайте ваші ключі:
```env
GITHUB_TOKEN=ваш_токен_github_models
SERPAPI_KEY=ваш_ключ_serpapi
INPUT_FILE=products.txt
OUTPUT_FILE=result.csv
```

---

## Отримання ключів

1. **GitHub Models**: Перейдіть до [GitHub Marketplace Models](https://github.com/marketplace/models), оберіть модель (наприклад, GPT-4o mini) та згенеруйте **Personal Access Token**.
2. **SerpApi**: Зареєструйтесь на [SerpApi](https://serpapi.com/) та скопіюйте ваш API Key з панелі керування.

---

## Запуск

1. Підготуйте файл `products.txt` (один рядок — один продукт).
2. Запустіть скрипт:
```bash
python main.py
```

## Технологічний стек

* **Python 3.10+**
* **httpx** — асинхронні HTTP запити.
* **SerpApi** — Google Search API для отримання результатів.
* **OpenAI SDK** — доступ до моделей через GitHub Inference API.
* **Pydantic** — сувора типізація даних.

---

## Обмеження

* Швидкість роботи залежить від лімітів SerpApi (безкоштовний план має обмеження на кількість запитів).
* Для стабільного вибору конкретного товару рекомендується використовувати чіткі назви у `products.txt`.
